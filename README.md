### AI Recipes

This app allows you to generate recipes from a picture of the ingredients you have available, using AI.

## Getting Started

Run the development server:

```bash
npm run dev
```

This app is using Capacitor :

Run the Android app :

```bash
npx cap open android
```

Run the iOS app (with XCode):

```bash
npx cap open iOS
```

```bash
npx cap sync
```
